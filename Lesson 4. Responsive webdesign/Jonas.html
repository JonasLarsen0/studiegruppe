In your study groups (help each other code individually):

1. Create a one-page website containing several sections of content. The page could represent a blog, a news site, etc.
2. For text content, you may use lorem ipsum ( https://en.lipsum.com/) or add your own content.
3. The page as a whole must be made responsive using a combination of the techniques we have talked about today (i.e. viewport, CSS grids, and media queries).
4. At the top of the page, create a card grid (using CSS grids). Each card must act as a teaser for a corresponding section of content on the page. (For inspiration, try following the tutorial in: Robinson, B., 2017).
5. Each card should contain a headline (matching the corresponding section) as well as a short teaser text (and if possible an image). Each card should also include an internal link to the corresponding section - allowing users to quickly navigate to that section.
